Initial release of the TechDocs Editor plugin suite, which enables in-app documentation editing directly in Backstage. Features include a WYSIWYG and Markdown editor powered by Toast UI Editor, a file tree panel for multi-file navigation, and pull/merge request submission to GitHub and GitLab repositories.
