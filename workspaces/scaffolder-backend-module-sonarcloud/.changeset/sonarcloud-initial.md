New plugin: 5 scaffolder actions for SonarCloud project onboarding

- `sonarcloud:create-project` — Create project with Bearer auth and org support
- `sonarcloud:bind-project` — ALM binding (GitHub, GitLab, Bitbucket, Azure)
- `sonarcloud:set-default-branch` — Rename default branch (defaults to `main`)
- `sonarcloud:set-quality-gate` — Assign quality gate by name
- `sonarcloud:set-new-code-definition` — Configure new code period
