# ArgoCD Node Plugin

This plugin provides the ArgoCDService class and related utilities for interacting with ArgoCD APIs from backend services.

## Features

- ArgoCDService class for managing ArgoCD resources
- Utility functions for ArgoCD API interactions
- Type definitions for ArgoCD operations

## Installation

```bash
yarn add @backstage-community/plugin-argocd-node
```

## Usage

```typescript
import { ArgoCDService } from '@backstage-community/plugin-argocd-node';
```
