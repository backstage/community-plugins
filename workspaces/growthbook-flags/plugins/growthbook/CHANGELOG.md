# @backstage-community/plugin-growthbook

## 0.1.0

### Minor Changes

- Initial release of GrowthBook feature flags plugin for Backstage
- Display feature flags in entity pages with type badges and default values
- Project filtering via button group populated from GrowthBook management API
- Alphabetical sorting of flags
- JSON detail dialog for complex flag values
- Custom entity page layout (Overview, Docs, Feature Flags only)
- Support for `growthbook.io/enabled`, `growthbook.io/env`, and `growthbook.io/project` annotations
