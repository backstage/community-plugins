New plugin: 5 scaffolder actions for SonarCloud project onboarding

- `sonarcloud:createProject` — Create project with Bearer auth and org support
- `sonarcloud:bindProject` — Bind to GitHub repo via v2 API
- `sonarcloud:setDefaultBranch` — Rename default branch (defaults to `main`)
- `sonarcloud:setQualityGate` — Assign quality gate by name
- `sonarcloud:setNewCodeDefinition` — Configure new code period
