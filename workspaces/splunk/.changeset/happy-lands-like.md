**Migrated Splunk On-Call plugin styling from Material-UI (MUI) to Backstage UI (BUI).** All components now use Backstage's design system with CSS modules for styling, providing improved theming consistency and reduced dependencies. The standalone SplunkOnCallPage route now displays an informative EmptyState directing users to the catalog, where the Splunk On-Call card integration is intended to be used.
