# Analytics Workspace

This workspace contains plugins for integrating various analytics providers into Backstage.

## Plugins

- [analytics-module-ga4](./plugins/analytics-module-ga4): Analytics module for Google Analytics 4.
- [analytics-module-matomo](./plugins/analytics-module-matomo): Analytics module for Matomo.
- [analytics-module-newrelic-browser](./plugins/analytics-module-newrelic-browser): Analytics module for New Relic Browser.
- [analytics-provider-segment](./plugins/analytics-provider-segment): Analytics provider for Segment.
