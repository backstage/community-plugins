# Azure DevOps Workspace

This workspace contains plugins for integrating Azure DevOps into Backstage.

## Plugins

- [azure-devops](./plugins/azure-devops): Frontend plugin to view Azure DevOps builds and pull requests.
- [azure-devops-backend](./plugins/azure-devops-backend): Backend plugin logic for Azure DevOps integration.
- [azure-devops-common](./plugins/azure-devops-common): Shared code for Azure DevOps plugins.
- [scaffolder-backend-module-azure-devops](./plugins/scaffolder-backend-module-azure-devops): Scaffolder module for Azure DevOps actions.
- [search-backend-module-azure-devops](./plugins/search-backend-module-azure-devops): Collator to index Azure DevOps resources into search.
